Cloud Computing


Mastering the core skill sets required to design and deploy dynamically scalable and reliable applications on popular cloud platform providers like Microsoft Azure, Amazon Web Services (AWS), and Google Cloud Platform (GCP).

Learning Outcome:

  • Understand foundational and core concepts of cloud computing and cloud security
  • Build expertise in implementing and maintaining cloud infrastructure
  • Master the core skill sets required to design and deploy dynamically scalable and reliable applications on popular cloud platform providers like Microsoft Azure, AWS, and GCP

Topics: Cloud Automation, Database, Linux, CI/CD, Networking, DevOps Foundation, Development, Automates operational tasks of container management, Containerization, Cloud Development Management.

Employment Opportunities: Cloud Engineer, Cloud Architect, Researchers.

Eligibilities:
